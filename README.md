Rolling
======================
롤링 컴포넌트<br>
목록형 데이터를 상하 또는 좌우로 롤링시키며 보여주는 컴포넌트 입니다.

## Feature
* 순환Rolling, 비순환Rolling
* 롤링 방향을 좌우, 상하 지원
* 롤링 속도 조절
* 이전페이지, 다음페이지, 특정페이지로 이동
* 이동시 애니메이션 효과 (정속, 가감속)
* 커스텀이벤트(페이지 이동 전, 페이지 이동 후)
