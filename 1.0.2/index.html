    Rolling

    Rolling Component
    Show data list and move the list items.

    Feature

    • Circular Rolling, Incircular acyclic rolling
    • Move panel each directions(left and right / up and down)
    • Set duration
    • Move to specific page
    • Animation effect
    • Custom effect
